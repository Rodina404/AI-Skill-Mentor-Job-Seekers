- Throughput: **1,500-2,000 req/sec per worker**

- Job recommendation: **O(n)** linear time
- User-based CF: **O(u + k×c)** efficient

## 📝 USAGE INSTRUCTIONS

### Quick Start (5 minutes)
```bash
1. cd graduation_project
2. pip install -r requirements.txt
3. python quick_start.py
```

### Start API Server
```bash
python main_app/app.py
# Visit http://localhost:8000/docs
```

### Run Tests
```bash
pytest tests/test_system.py -v
```

### Make Recommendations
```bash
curl -X POST http://localhost:8000/api/v1/recommend-job \
  -H "Content-Type: application/json" \
  -d '{"skills": ["python", "ml"], "top_n": 5}'
```
